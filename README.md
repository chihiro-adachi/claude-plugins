# claude-plugins

Claude Code 用のプラグイン集です。

## プラグイン一覧

| プラグイン | 説明 |
| --- | --- |
| [claude-changelog](plugins/claude-changelog/) | Claude Code の変更履歴を確認する |
| [codex-plan-review](plugins/codex-plan-review/) | 実装計画を Codex CLI でレビューする |

## インストール

```
/plugin marketplace add chihiro-adachi/claude-plugins
/plugin install claude-changelog@chihiro-adachi-claude-plugins
```
