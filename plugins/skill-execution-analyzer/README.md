# skill-execution-analyzer

スキル実行時のエラーや問題点を解析するプラグイン。

## できること

- セッショントランスクリプト（jsonl）と SKILL.md を照合し、実行時の問題を検出
- ツール呼び出しの失敗やエラーの特定
- SKILL.md で定義されたステップの実行順序の逸脱を検出
- スキップされたステップの検出
- 改善案の提示

## 使い方

`/skill-execution-analyzer` を実行、または自然言語で質問する。

```
/skill-execution-analyzer
```

```
さっき実行したスキルを解析して
```

```
skill-execution-analyzer で claude-changelog スキルの実行ログを確認して
```

## オプション

- **スキル名**: 解析対象のスキル名を指定（省略時は直近に実行されたスキルを推定）
- **セッション ID / ファイルパス**: 解析対象のトランスクリプトを指定（省略時は直前のセッションを候補とする）
